# Global file and folder

- Applies to all projects unless overridden.

## Session code changes

- Every session has there own code changes, it not take last turn change from another session.
- but we connect with cloud repo with github/gitlab it show all current changes on code base.

- Last turn changes: session changes
- Git changes: all change from last commit
